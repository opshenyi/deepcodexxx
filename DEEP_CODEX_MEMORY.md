## Architecture Decisions

- Product code is clean-room TypeScript. Reference repos are study material only.
- Default DeepSeek base URL is `https://api.deepseek.com`.
- Default model is configurable through `DEEPSEEK_MODEL` and currently falls back to `deepseek-v4-flash`, matching official DeepSeek V4 OpenAI-format model docs checked on 2026-06-09. Legacy `deepseek-chat` and `deepseek-reasoner` aliases are no longer template defaults because official docs schedule them to stop working on 2026-07-24 15:59 UTC.
- Default DeepSeek V4 thinking mode is `disabled` for predictable local demos, but opt-in thinking mode now preserves and replays DeepSeek `reasoning_content` across tool-call turns inside the provider transcript.
- Agent tools are evented and policy-controlled.
- Workspace memory lives in `.deepcodex/memory.md`.
- Product memory for future Codex sessions lives in this file.
